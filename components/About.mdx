# Moléculas (Molecules)

Moléculas são grupos relativamente simples de elementos de IU que funcionam juntos como uma unidade. Por exemplo, uma label de formulário, search input e botão podem se unir para criar uma molécula de campo de busca.

Molecules are relatively simple groups of UI elements that work together as a unit. For example, a form label, search input, and button can combine to form a search field molecule.
